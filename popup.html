Experiencing issues with the modal?

You can utilize the Reset function to restore it to its functional state.

Are you sure you want to proceed with this process?

Please note that it will result in the deletion of all saved data, including:

* custom shortcuts
* custom objects
* saved organizations
* favorites.

 Additionally, all preferences will be reset.
To open the modal in the future, you will need to use the default Ctrl+Q.
If you wish to change this setting, you can do so by going to the Settings page within the modal itself.
